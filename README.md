# Pollution Awareness & Solutions Web App
This website contains alot of information regarding thebiggest problem of the world,pollution. The problems caused by it and the solutions are refered in this website.

##  Features

* **Educational Content:** I gave clear explanation on what pollution is,it's progress referring Lahore and solutions to control it.
* **Multimedia Integration:** Related video and pictures are compiled for better understanding of user.
* **Styling of web**          Distinctive features from style.css are used to make web beautiful.

## Repository Structure


 index.html          # Main opening page

 lahore.html         # Case study page (Lahore - Most Polluted City of Pakistan)
 sol.html            # Solutions & Important Measures page
